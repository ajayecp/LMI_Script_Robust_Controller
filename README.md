# Robust Controller Design for a Chemical Reactor

# Requirements before executing
1. Install Python
2. Install Git

# Steps to reproduce
1. Clone this repository.
2. git clone https://github.com/ajayecp/LMI_Script_Robust_Controller

# Option 2
2. Download .zip file in the link and extract in your directory

3. Open the project folder.
4. Creat virtual environment
5. python -m venv env
6. activate using .\env\Scripts\activate
7. install requirements using pip install -r requirements.txt
8. run the code python .\Seminario_H_Inf.py
